# Number Guessing Game (Java)

## 📌 Description
A console-based number guessing game where the user attempts to guess a randomly generated number within a given range. This project focuses on loops, conditionals, and user interaction in Java.

## 🚀 Features
- Generates a random number within a defined range
- Provides feedback if the guess is too high or too low
- Tracks the number of attempts
- Continues until the correct number is guessed

## 🛠️ Built With
- Java
- Random (number generation)
- Scanner (user input)

## ▶️ How to Run
1. Clone the repository
2. Open in any Java IDE (IntelliJ, Eclipse, etc.)
3. Run `Main.java`
4. Enter guesses in the console until correct

## 📚 What I Learned
- Using loops (`do-while`) for repeated user interaction
- Implementing conditional logic for feedback
- Generating random numbers with `Random`
- Tracking state using variables (attempt counter)

## 🔮 Future Improvements
- Add difficulty levels (easy, medium, hard)
- Limit number of attempts
- Add replay option after winning
- Validate input to prevent crashes from non-numeric values
